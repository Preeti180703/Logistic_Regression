# Logistic_Regression
A machine learning project based on Logistic Regression for binary classification. The repository covers data cleaning, feature selection, model training, performance evaluation (accuracy, confusion matrix), and result interpretation using Python libraries such as NumPy, Pandas, and Scikit-learn.

# Methodology:
1. Import required libraries
2. Load the dataset
3. Data cleaning and preprocessing
4. Feature selection
5. Train-test split
6. Train Logistic Regression model
7. Model evaluation
   
#  Model Evaluation:
- Accuracy Score
- Confusion Matrix
- Classification Report
